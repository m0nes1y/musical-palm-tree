# Agent 上架信息

## 基本信息
- **名称**：Web Scraper Pro（精准爬虫师）
- **副标题**：自然语言驱动 · Markdown+JSON双格式输出
- **分类**：效率工具 / 数据采集
- **标签**：爬虫、网页抓取、数据提取、Crawl4AI、Scrapling、多页抓取

## 简短描述（≤30字）
用自然语言描述抓取目标，AI自动探索页面并精准提取数据。

## 详细描述（≤200字）
融合Crawl4AI（整页渲染+智能去重）和Scrapling（CSS/XPath/正则精准提取），支持单页、多页翻页、列表→详情页等复杂场景。用户用自然语言描述需求，Agent自动完成：页面分析→选择器构建→内容提取→Markdown+JSON双格式输出。遇反爬或失败自动说明原因和降级建议。

## 触发词示例
- "帮我抓取这个页面所有文章标题"
- "抓取某电商网站的商品名称和价格"
- "提取这个页面所有图片的src"
- "爬取这个论坛的所有帖子标题和作者"
- "抓取某新闻网站最近一周的头条新闻"

## 注意事项
- 仅抓取公开可访问网页
- 不支持登录/付费内容
- 尊重目标网站访问频率限制
